# Hulu Clone

This is a Hulu Clone made with NextJs and TailwindCSS

## Link to view the clone

https://hulu2-0-clone.vercel.app/

Steps to use

- cd into the app

```bash
cd hulu2.0-clone
```

- If you use yarn then run

```bash
yarn run dev
```

and if you use npm delete yarn.lock and run

```bash
npm run dev
```
